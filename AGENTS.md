# Agent Instructions

## Git Commit Format

Use this commit message format:

```text
type(scope): title
```

- `type` is a lowercase change category, such as `feat`, `fix`, `docs`, `chore`, `refactor`, `test`, `build`, or `ci`.
- `scope` is the lowercase area changed, such as `app`, `electron`, `repo`, or `settings`.
- `title` is a concise imperative summary of the change.
